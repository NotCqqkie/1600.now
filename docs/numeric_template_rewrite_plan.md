# Numeric-template Rewrite Plan (PAST bank only)

Input: `scripts/numeric_template_work.json` (generated by `scripts/find_numeric_templates.mjs`).

## HARD RULE

**Only rewrite PAST-bank questions.** The unofficial bank must NOT be content-rewritten under any circumstance, even if near-duplicates exist there. The generator enforces this by omitting the unofficial bank from the plan.

## What's in scope

Each entry targets one PAST-bank question that:

- shares a number-normalized signature with at least one sibling (same skeleton, different numbers);
- has NO image associated (checked against `questionImageMap` and question fields);
- is NOT the kept anchor for its group.

For every group, one non-imaged anchor (or every imaged anchor) is kept untouched; all OTHER non-imaged members are rewritten.

## Worker prompt (per question)

For bank `<bank>` id `<id>` in `src/data/questions/<bank>.json`:

1. Load the question text.
2. Preserve: every numeric value, the correct answer, the answer choices' numeric content, category/domain/skill, and all math notation (LaTeX delimiters, `$...$`, etc.).
3. Rewrite: the surface context/story so it does not read as a near-duplicate of its siblings (listed in `nonImageIds` / `withImageIds`). Pick a plausible distinct real-world scenario that yields the same math.
4. Keep length within ±25% of the original.
5. Commit per question: `[codex] Template rewrite id:<id> (<bank>)`.

## Constraints

- Never touch a question in `withImageIds`.
- Never touch the kept anchor for a group.
- Never change numeric values or the correct-answer mapping.
- Never rewrite questions from `unofficialQuestions.ts`.
